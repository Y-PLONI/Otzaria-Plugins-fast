# 📦 Otzaria Plugin Studio 🚀

בניתם כלי מדהים לאוצריא ב-HTML? מזל טוב. הסטודיו כאן כדי לוודא שתוכלו לארוז אותו ולחזור ללמוד כמה שיותר מהר. 

הכלי דואג לכל ה"מסביב": יצירת המניפסט, ניהול הרשאות API, והפיכת התוסף לזמין ב-Offline על ידי אריזת משאבים חיצוניים. 

## 🛠️ מה הסטודיו עושה בשבילך?
* **אריזה ב-3 קליקים:** זורקים HTML, מקבלים `.otzplugin`.
* **סנכרון עיצוב אוטומטי:** מזריק קוד שדואג שהתוסף שלכם ייראה מעולה גם במצב לילה.
* **מקומי נטו:** המערכת עוזרת לכם להוריד סקריפטים ועיצובים מרוחקים כדי שהתוסף לא יהיה תלוי באינטרנט.
